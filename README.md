# Pig Latin Converter
# Purpose: This command line application allows the user to convert an English word into Pig Latin and vice versa. It is essentially a rearrangement of words that alrady exist by following a defined set of rules: 
  # 1. For words that start with a consonant, the consonant is moved to the end of the word and "ay" is added after it.
  # 2. For word beginning with a vowel, "way" or "yay" is added to the end of the word.
# Installation: To run this program, Visual Studio Code and the latest version of Python is required.
