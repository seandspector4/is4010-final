# Pig Latin Converter
# Purpose: This command line application allows the user to convert an English word into Pig Latin. It is essentially a rearrangement of words that alrady exist by following a defined set of rules: 
  # 1. For words that start with a consonant, the consonant is moved to the end of the word and "ay" is added after it.
  # 2. For word beginning with a vowel, "yay" is added to the end of the word.
# Installation and Usage: To run this program, Visual Studio Code and the latest version of Python is required.
# Examples: If the word "Read" is used as input, the Pig Latin output would be "eadRay". If the word "Art" is used as input, the Pig Latin output should be "Artyay".
